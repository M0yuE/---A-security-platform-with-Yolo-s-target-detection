# 目标检测和安防平台的整合应用

#### 介绍
springboot+redis+esp8266+红外烟雾传感器+yolov5+echarts数据大屏


#### 安装教程

1. Node.js v14.17.4.
2. java version "1.8.0_341"
3. yolo v5
4. torch环境在requestments.txt中
